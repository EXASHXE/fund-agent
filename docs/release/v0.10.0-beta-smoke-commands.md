# v0.10.0 Beta Smoke Commands

## Default Test Gate

```bash
PYTHONPATH=. pytest -q
```

## CLI Smoke

```bash
fund-agent --help
fund-agent doctor --help
fund-agent regressions --pretty
```

## Portfolio Analysis

```bash
fund-agent analyze-portfolio --input examples/user_portfolio_templates/fund_portfolio_input_demo.json --pretty
fund-agent analyze-portfolio --input examples/user_portfolio_templates/fund_portfolio_input_demo.json --format markdown --output /tmp/fund-agent-demo-report.md
```

## Report Rendering

```bash
fund-agent render-report --input final_report.json --format markdown --output report.md
```
