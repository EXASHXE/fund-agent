# v0.9.4 Markdown Report Notes

## Deterministic Markdown Renderer

`src/skills_runtime/workflow/markdown_report.py` provides:
- `render_advisory_report_markdown(report, locale="zh-CN")` — renders structured report to zh-CN markdown

## zh-CN Report Sections

1. 直接回答
2. 组合概览
3. 当前主要风险
4. 持仓诊断
5. 证据状态
6. 数据缺口与限制
7. 操作边界
8. 建议下一步
9. 决策说明
10. 风险提示

## Report Mode Behavior

- Report-only: states no formal decision was generated
- Soft action advice: states no formal decision, provides guidance
- Formal decision: includes action table, evidence anchors, fee constraints
- Blocked decision: explains blockers with reason codes

## CLI Support

```bash
fund-agent render-report --input final_report.json --format markdown --output report.md
```
