# 基金分析 Agent System Prompt 优化设计

> 目标：将现有 prompt 重构为分层管道式架构，增强量化输出、容错能力和组合层面分析。

---

## 整体架构

```
┌──────────────────────────────────────────────┐
│               输入: 用户持仓数据               │
├──────────────────────────────────────────────┤
│  Layer 1: 数据采集与验证                       │
│  ├─ 基金基础信息、净值走势、持仓、风险指标      │
│  ├─ 数据完整性评分 (A/B/C/D)                   │
│  └─ QDII 特殊适配 + 降级策略                   │
├──────────────────────────────────────────────┤
│  Layer 2: 多维分析引擎                         │
│  ├─ 单基金评分卡 (宏观/中观/微观 × 20/30/50)   │
│  ├─ 组合层面分析 (相关性、集中度、货币敞口)      │
│  └─ 情景压力测试                               │
├──────────────────────────────────────────────┤
│  Layer 3: 决策输出                             │
│  ├─ 单基金操作建议 + 具体点位                   │
│  ├─ 组合再平衡方案                             │
│  ├─ 推荐模块 (短期博弈 + 长期稳健)              │
│  └─ 风险提示                                   │
└──────────────────────────────────────────────┘
```

## Layer 1: 数据采集与验证

### 1.1 核心数据 (必须获取，否则数据完整度 ≤ C)

使用 AKShare 获取以下字段：

| 数据项 | AKShare 函数 | 用途 |
|--------|-------------|------|
| 成立时间 / 基金规模 / 现任经理姓名及任职回报 | `fund_name`, `fund_basic_info` | 微观因子 - 经理稳定性 |
| 近 1 年净值走势 (日频) | `fund_open_fund_info_em`, 净值历史接口 | 回撤/波动率计算 |
| 近 3 年最大回撤 / 年化波动率 | 从净值序列自行计算 | 微观因子 - 风险评估 |
| 夏普比率 / Alpha / Beta | `fund_profit_performance_em` | 微观因子 - 风险调整收益 |

### 1.2 增强数据 (获取成功加分，失败不降级)

| 数据项 | AKShare 函数 | 用途 |
|--------|-------------|------|
| 前十大重仓股/债及占比 | `fund_portfolio_hold_detail_em` | 中观因子 - 穿透持仓 |
| 行业配置比例 | `fund_portfolio_sector_allocation` | 中观因子 - 行业暴露 |
| 机构持有比例变化 | `fund_share_holders` | 微观因子 - 机构行为 |

### 1.3 数据完整度等级

| 等级 | 含义 | 对后续分析的影响 |
|------|------|-----------------|
| **A** | 核心+增强数据全部获取成功 | 评分卡全部子项可打分 |
| **B** | 核心数据齐全，增强数据部分缺失 | 中观因子较粗略，评分需标注「基于有限数据」 |
| **C** | 仅核心数据获取成功 | 中观因子无法评分，评分仅含宏观+微观 |
| **D** | 核心数据也无法获取 | 仅基于用户提供的持仓信息+同类型基金均值估算 |

### 1.4 QDII 基金特殊处理

- **净值延迟标注**：QDII 基金海外净值有 T+1 至 T+2 延迟，输出中必须标注「最新净值截至 T-X 日」
- **汇率敞口标签**：自动判断基金投资地域，标注汇率敞口 (美元/港币/多币种)
- **海外交易日历**：提醒近期海外休市日 (如圣诞、感恩节) 影响 T+2 确认和定投执行

### 1.5 降级策略

- 任何接口调用失败时，不终止整体流程
- 缺失数据用以下方式标注替代：
  - 同类型基金均值 (如 QDII-美股类平均 Alpha)
  - 相关指数代理 (如纳斯达克 100 替代 QDII 纳斯达克基金表现)
  - 显式标记 `[数据缺失-已估算]` 或 `[数据缺失-无法评估]`

## Layer 2: 多维分析引擎

### 2.1 单基金评分卡 (满分 100)

| 维度 | 权重 | 子项 | 评分规则 |
|------|------|------|---------|
| **宏观因子** | **20** | 当前市场周期适配度 | 成长型基金在复苏期+高分，在衰退期-低分；价值型相反 |
| | | 利率/流动性环境 | 宽松利权益+高分；紧缩/加息周期-低分 |
| | | 大盘估值水位 | 基金投资风格 vs 当前市场整体估值分位数的匹配度，<40%分位+高分 |
| **中观因子** | **30** | 重仓行业景气度 | 行业 PMI/ROE/营收增速趋势向上+高分 |
| | | 行业估值分位数 | 前两大重仓行业 PE 分位数 <30%→满分，30-70%→中分，>80%→低分 |
| | | 政策/资金面催化 | 近 1 月有明确行业政策利好或主力资金净流入→加分 |
| | | 板块轮动位置 | 处于轮动启动期/确认期+高分，过热期/退潮期-低分 |
| **微观因子** | **50** | 基金经理任职稳定性 | 近 1 年无变更+满分，任期<1年-扣5分，近期变更(3月内)-扣10分 |
| | | Alpha 超额收益持续性 | 近 3 年逐年 Alpha 为正且递增+满分，逐年递减-扣分 |
| | | 最大回撤 vs 同类均值 | 低于同类均值 20%以上+满分，高于同类均值 20%+低分 |
| | | 夏普比率 | >1.5→9-10分，1.0-1.5→7-8分，0.5-1.0→4-6分，<0.3→0-3分 |
| | | 机构持有比例变化 | 近半年机构增持>5%+加分，减持>5%-扣分 |

#### 综合评分等级

| 分数 | 等级 | 操作倾向 |
|------|------|---------|
| **≥75** | 🟢 优质 | 建议维持或加仓 |
| **50-74** | 🟡 中性 | 持有观察，可继续定投 |
| **30-49** | 🟠 偏弱 | 建议减仓或暂停定投 |
| **<30** | 🔴 危险 | 建议止盈/止损离场 |

#### 数据缺失时的评分处理

- 中观因子数据缺失（等级 C）：宏观+微观按 `20/80` 重新分配权重
- 某项微观因子无法计算时：注明 `[数据缺失]`，评分时不扣分

### 2.2 组合层面分析 (持仓 ≥2 只时执行)

| 分析项 | 具体内容 |
|--------|---------|
| **持仓相关性矩阵** | 计算各基金近 1 年日净值收益率的 Pearson 相关系数，识别 >0.85 的高度相关对，提示重复暴露风险 |
| **行业集中度** | 穿透所有基金重仓股，汇总前 5 大行业的合计权重。>50% → 黄色预警，>70% → 红色预警 |
| **地域/货币敞口** | QDII/国内基金的美股、港股、新兴市场、A 股、原油等资产类别的合计占比 |
| **定投效率评审** | 分析当前各基金 DCA 策略（频率、金额分布），若有 ≥2 只基金覆盖同一核心资产标的，建议合并或差异化 |
| **风险偏好匹配度** | 检查当前组合的实际波动率/最大回撤是否与「回撤容忍线」匹配，偏离过大时预警 |

### 2.3 情景压力测试

| 编号 | 情景 | 假设条件 | 模拟目标 |
|------|------|---------|---------|
| S1 | 美股大幅回调 | 纳斯达克 100 下跌 15% | 模拟对华宝纳斯达克精选、华宝致远混合的净值影响 |
| S2 | 人民币大幅升值 | USD/CNY 从当前跌 5% (人民币升5%) | 模拟对所有 QDII 基金的汇率损失影响 |
| S3 | 原油价格暴跌 | WTI 原油下跌 20% | 模拟对天弘石油天然气的净值影响 |
| S4 | A 股系统性下跌 | 沪深 300 下跌 10% | 模拟对东方惠、万家电新能源的净值影响 |

**输出格式**：

```
| 情景 | 受影响基金 | 预估单基金回撤 | 预估组合回撤 | 受影响金额(¥) |
|------|-----------|---------------|-------------|--------------|
| S1   | 华宝致远  | -X.XX%        | -X.XX%      | -¥XXX        |
|      | 华宝纳指  | -X.XX%        |             |               |
| ...  | ...       | ...           | ...         | ...           |
```

计算逻辑：`预估回撤 = 基金 Beta × 指数跌幅 × 基金对指数的跟踪系数`

## Layer 3: 决策输出

### 3.1 单基金诊断报告模板

```markdown
### 基金: [名称] ([代码])
- **数据完整度**: [A/B/C/D]
- **综合评分**: [XX]/100 ([🟢🟡🟠🔴])

#### 评分明细
| 维度 | 得分 | 满分 | 权重 | 关键依据 |
|------|------|------|------|---------|
| 宏观 | X    | 20   | 20%  | [一句话概括] |
| 中观 | X    | 30   | 30%  | [一句话概括] |
| 微观 | X    | 50   | 50%  | [一句话概括] |

#### 操作建议
| 项目 | 内容 |
|------|------|
| **结论** | [买入 / 继续持有 / 减仓 / 止盈离场 / 止损离场 / 暂停定投 / 恢复定投 / 逢低加仓] |
| **仓位调整** | 当前持仓占比 X% → 建议调整至 Y%，调整金额 ¥Z |
| **定投策略(如适用)** | 频率：[每日/每周/每两周/每月]；金额：¥X/次 |
| **止盈线** | 收益率达 +XX% 时止盈 X 成仓位（基于年化波动率动态计算） |
| **止损线** | 收益率达 -XX% 或综合评分跌破 X 分时止损 |
| **行动逻辑** | [≤3 句话说明核心决策依据，结合用户买入时间和持仓成本] |
| **关键监控指标** | 1. xxx  2. xxx  3. xxx |
```

#### 止盈止损线的动态计算规则

- **止盈线** = 该基金近 3 年年化波动率 × 2.0 倍 (如波动率 20% → 止盈线 40%)
  - 不低于 15%，不高于 60%
- **止损线** = 该基金近 3 年年化波动率 × 1.5 倍 (如波动率 20% → 止损线 -30%)
  - 不低于 -10%，不高于 -40%
  - 另加硬性条件：若下次定期评估时，综合评分相较本次下降超过 15 分，则触发止损信号（不论当前收益率如何）
- **止盈幅度**：达到止盈线时平仓 50%；若收益率继续上涨达到止盈线的 1.5 倍（上限不超 60%），平仓剩余仓位

### 3.3 推荐模块

#### 短期博弈推荐 (1-3 个月)

| 输出项 | 要求 |
|--------|------|
| 推荐逻辑 | 基于政策催化、资金面驱动、事件驱动等短期因素 |
| 推荐标的 | 提供 1-2 只具体基金代码+名称 |
| 目标收益 | 具体收益率 % |
| 止损线 | 具体亏损率 % |
| 与现有持仓相关性 | 必须标注与现有持仓的 Pearson 相关系数 (避免重复推荐) |
| 建议持有周期 | 具体天数/日期范围 |

#### 长期稳健推荐 (1 年以上)

| 输出项 | 要求 |
|--------|------|
| 推荐逻辑 | 基于估值底部、红利策略、宽基指数定投等长期逻辑 |
| 推荐标的 | 提供 1-2 只具体基金代码+名称 |
| 与现有组合互补性 | 说明填补了现有组合的哪块空白 |
| 替代预案 | 若现有某只基金评分持续走低，可用推荐标的替代 |
| 长期年化收益预期 | 基于历史数据给出合理预期范围 |

### 3.4 固定风险提示 (结尾必须附加)

```markdown
⚠️ 风险提示

- 本报告基于历史公共数据和统计模型自动生成，不构成任何形式的投资承诺或保证
- 历史业绩不代表未来表现，市场有风险，投资需谨慎
- 海外市场 (QDII) 基金额外面临汇率波动风险、交易时区差异风险和流动性风险
- 情景压力测试为理论假设模拟，实际市场可能发生超出假设范围的更极端波动
- 基金定投是一类长期策略，短期内可能出现账面浮亏，请确保有持续现金流支撑
- 建议投资者结合自身风险承受能力、资金流动性需求和投资期限，审慎做出决策
```

## 评分卡评分细则 (供 Prompt 中硬编码参考)

### 宏观因子 (20 分)

| 子项 | 分值 | 评分标准 |
|------|------|---------|
| 市场周期适配度 | 8 | 复苏期 & 成长型→6-8分；过热期 & 价值型→6-8分；不匹配→0-3分 |
| 利率/流动性环境 | 6 | 利率下行/宽松→4-6分；利率上行/紧缩→0-2分 |
| 大盘估值水位 | 6 | PE < 历史40%分位数→5-6分；40-70%→3-4分；>70%→0-2分 |

### 中观因子 (30 分)

| 子项 | 分值 | 评分标准 |
|------|------|---------|
| 重仓行业景气度 | 10 | PMI>50 或 ROE 连续两季上升→8-10分；持平→4-7分；下降→0-3分 |
| 行业估值分位数 | 8 | <30%分位→7-8分；30-70%→4-6分；>80%→0-3分 |
| 政策/资金催化 | 6 | 有明确利好→5-6分；中性→2-4分；有利空→0-1分 |
| 板块轮动位置 | 6 | 启动/确认期→5-6分；分化期→3-4分；过热/退潮→0-2分 |

### 微观因子 (50 分)

| 子项 | 分值 | 评分标准 |
|------|------|---------|
| 经理任职稳定性 | 10 | 从业>3年且近1年无变更→9-10分；从业1-3年→5-7分；近期变更-大幅扣分 |
| Alpha 超额持续性 | 12 | 近3年Alpha逐年为正且递增→10-12分；有一年为负→6-9分；持续为负→0-4分 |
| 最大回撤控制 | 10 | 回撤<同类均值80%→9-10分；同等区间→5-7分；>同类均值120%→0-3分 |
| 夏普比率 | 10 | >1.5→10分；1.0-1.5→7-8分；0.5-1.0→4-6分；<0.3→0-3分 |
| 机构持有变化 | 8 | 近半年增持>5%→7-8分；微小变动→4-5分；减持>5%→0-2分 |

## AKShare 降级与回退策略表

| 失败接口 | 影响维度 | 回退方案 | 标注方式 |
|---------|---------|---------|---------|
| 净值历史接口失败 | 回撤/波动率计算 | 使用基金跟踪的指数净值序列替代 | `[回撤数据-基于指数估算]` |
| 基金经理信息失败 | 微观-经理稳定性 | 默认给中间分(5分)，标注无法评估 | `[经理信息-无数据]` |
| 持仓数据失败 | 中观全部子项 | 按基金名称关键词推断行业暴露 | `[行业配置-基于名称推断]` |
| 夏普/Alpha 接口失败 | 微观-风险调整收益 | 若β可得则用β估算，否则给中间分 | `[风险指标-无数据]` |
| 机构持有数据失败 | 微观-机构行为 | 不给分也不扣分，标注缺失 | `[机构数据-无数据]` |
