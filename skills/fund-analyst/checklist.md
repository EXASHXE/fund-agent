# 输出自检清单

生成完整报告后，**必须**逐项验证以下清单，确保无遗漏。

## 逐基金检查（每只基金）

- [ ] 是否输出了数据完整度等级（A/B/C/D）？
- [ ] 完整度为 C/D 时，是否标注了 `[数据缺失-xxx]` 标记？
- [ ] 是否为 QDII 基金额外输出了「最新净值截至 T-X 日」和「汇率敞口」？
- [ ] 评分表是否包含宏观/中观/微观三项，且权重标注正确（20/30/50 或 C 级的 20/80）？
- [ ] 综合评分是否带颜色标签（🟢🟡🟠🔴）？
- [ ] 「操作建议」表是否完整包含：结论、仓位调整、定投策略、止盈线、止损线、行动逻辑、关键监控指标？
- [ ] 止盈线和止损线是否基于年化波动率×2.0 / ×1.5 计算出**具体百分比数字**（非占位符）？
- [ ] 止盈线是否在 15%-60% 范围内？止损线是否在 10%-40% 范围内？
- [ ] 行动逻辑是否结合了用户的实际买入时间、持仓成本、当前收益率？
- [ ] 关键监控指标是否为 2-3 个具体可跟踪的变量（非空泛描述）？

## 组合分析检查（持仓 ≥2 只时）

- [ ] 是否输出了持仓相关性矩阵（含 Pearson r 值）？
- [ ] 是否标记了 >0.85 的高度相关配对？
- [ ] 是否输出了行业集中度分析（前 5 大行业合计权重）？
- [ ] > 50% 是否黄色预警？>70% 是否红色预警？
- [ ] 是否输出了地域/货币敞口汇总？
- [ ] 是否进行了定投效率评审？
- [ ] 是否输出了情景压力测试表（4 类情景 × 受影响基金）？
- [ ] 情景测试表是否包含「影响金额(¥)」列？

## 推荐模块检查（如有新推荐）

- [ ] 短期推荐是否标注了与现有持仓的 Pearson 相关系数？
- [ ] 短期推荐是否给出了目标收益率和止损线（具体数字）？
- [ ] 长期推荐是否说明了与现有组合的互补性？
- [ ] 长期推荐是否给出了替代预案（可替换哪只现有基金）？
- [ ] 推荐的基金代码和名称是否准确（非编造）？

## 格式完整性检查

- [ ] 报告末尾是否附加了**完整**的风险提示文本（6 条，不得省略或缩略）？
- [ ] 所有表格的列是否对齐？Markdown 表格是否正常渲染？
- [ ] 是否存在占位符（如 `X%`、`¥XXX`、`[待填]`）未被替换为实际数据？
- [ ] 所有百分比和金额是否带单位？

## 数据可靠性检查

- [ ] 所有从 AKShare 取到但存疑的数据是否标注了来源？
- [ ] 通过估算/替代得到的数据是否标注了 `[数据缺失-已估算]`？
- [ ] 完全无法获取的数据是否标注了 `[数据缺失-无法评估]` 而非静默跳过？
- [ ] QDII 基金的净值日期是否为 T-1 或 T-2？是否醒目标注？

## 逻辑一致性检查

- [ ] 评分等级与操作结论是否一致（≥75 却建议卖出为矛盾）？
- [ ] 仓位调整建议是否与总分匹配（低分加仓为矛盾）？
- [ ] 同一天分析的多只基金，宏观评分是否一致（宏观环境相同）？
- [ ] 是否存在同一资产被多只基金重复定投且未被指出？

## 新闻分析检查

- [ ] 是否为每只持仓基金采集了新闻？
- [ ] 是否输出了市场情绪总览（总新闻数、整体情绪、综合判断）？
- [ ] 逐基金分析是否包含情绪均值、正面率、负面率？
- [ ] 是否显示了情绪趋势（最近 7 天逐日变化）？
- [ ] 是否针对每只基金给出了情绪解读？
- [ ] 是否显示了新闻-净值相关性指标？

---

> 如任一项未通过，修正后重新验证全部清单。
