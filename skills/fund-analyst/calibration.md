# 评分校准用例

以下为 3 只典型基金的模拟数据与期望评分范围。agent 在初次分析前应先做自我校准，确保自己属于「正常评分者」（评分不极端偏高或偏低）。

---

## 校准用例 1：沪深 300 指数增强基金（基准锚）

### 假设数据

| 指标 | 数值 |
|------|------|
| 基金类型 | 股票指数增强型 |
| 成立时间 | 2015-03-15 |
| 规模 | 85 亿元 |
| 基金经理 | 张三，从业 8 年，管理本基金 5 年 |
| 近 3 年年化波动率 | 18.1% |
| 近 3 年最大回撤 | -28.5%（同类均值 -27.2%） |
| 近 3 年 Alpha（逐年） | +1.8% / +2.4% / +1.5% |
| Sharpe 比率（年化） | 0.72 |
| Beta | 0.96 |
| 机构持有变化 | 近半年增持 3% |
| 重仓行业 | 金融 22%、食品饮料 15%、电力设备 10% |

### 市场环境设定
- 当前经济周期：复苏初期
- 流动性：中性偏宽（LPR 小幅下调）
- 沪深 300 PE：处于历史 45% 分位

### 期望评分区间

| 维度 | 期望得分 | 允许范围 |
|------|---------|---------|
| 宏观 | 13-15 分 | 复苏期+指数增强风格匹配好；估值在 40-70% 中性区间 |
| 中观 | 17-20 分 | 金融+消费估值中性，无明显政策催化也无利空 |
| 微观 | 35-40 分 | Alpha 稳定为正但幅度小；回撤基本同类均值；Sharpe 0.72 中等；经理稳定 |
| **综合** | **65-75 分** | 🟡 偏上，接近 🟢 |

### 校准要点
- 如果综合分 <55 或 >85，说明打分尺度偏极，需调整
- 宏观分不应 <10（环境中性偏正）
- 微观分不应 >45（Alpha 空间小、Sharpe 中等）

## 校准用例 2：明星经理主动权益基金（高分锚）

### 假设数据

| 指标 | 数值 |
|------|------|
| 基金类型 | 偏股混合型 |
| 成立时间 | 2018-01-10 |
| 规模 | 220 亿元（百亿级） |
| 基金经理 | 李四，从业 12 年，管理本基金 6 年，金牛基金经理 |
| 近 3 年年化波动率 | 20.5% |
| 近 3 年最大回撤 | -22.1%（同类均值 -28.0%） |
| 近 3 年 Alpha（逐年） | +5.2% / +7.8% / +6.1% |
| Sharpe 比率（年化） | 1.35 |
| Beta | 0.88 |
| 机构持有变化 | 近半年增持 8% |
| 重仓行业 | 电子 25%、医药 18%、新能源 15% |

### 市场环境设定
- 当前经济周期：复苏中期
- 流动性：宽松（降准落地）
- 沪深 300 PE：处于历史 35% 分位（低估区间）

### 期望评分区间

| 维度 | 期望得分 | 允许范围 |
|------|---------|---------|
| 宏观 | 16-18 分 | 宽松+低估值，权益友好环境 |
| 中观 | 22-25 分 | 电子+新能源景气上行，医药 PE 低有修复预期 |
| 微观 | 43-48 分 | Alpha 持续超额+回撤显著低于同类+Sharpe 高+机构加仓+经理顶尖 |
| **综合** | **81-91 分** | 🟢 优质 |

### 校准要点
- 综合分应 ≥80（🟢）
- 微观分应 ≥40（各项微观指标都远优于同类）
- 如果微观分 <35，说明对 Alpha/Sharpe/回撤的评分尺度偏严厉

## 校准用例 3：高波动行业 ETF（低分锚）

### 假设数据

| 指标 | 数值 |
|------|------|
| 基金类型 | 被动指数型（行业 ETF 联接） |
| 成立时间 | 2020-06-01 |
| 规模 | 4.5 亿元（迷你基金） |
| 基金经理 | 王五，从业 3 年，管理本基金 1.5 年（IT 系统指增） |
| 近 3 年年化波动率 | 32.0% |
| 近 3 年最大回撤 | -48.0%（同类均值 -30.0%） |
| 近 3 年 Alpha（逐年） | -0.8% / -1.5% / -2.1%（持续为负=跟踪误差） |
| Sharpe 比率（年化） | 0.18 |
| Beta | 1.12 |
| 机构持有变化 | 近半年减持 6% |
| 重仓行业 | 有色金属 55%、化工 20%（高度集中） |

### 市场环境设定
- 当前经济周期：扩张后期
- 流动性：中性偏紧（货币净回笼）
- 周期板块 PE：处于历史 78% 分位（高估）

### 期望评分区间

| 维度 | 期望得分 | 允许范围 |
|------|---------|---------|
| 宏观 | 6-9 分 | 紧缩+高估值+扩张后期，不利环境 |
| 中观 | 8-12 分 | 有色 PE 高位，政策无催化，板块轮动近尾声 |
| 微观 | 12-18 分 | Alpha 负+回撤巨大+Sharpe 极低+机构减持+规模小 |
| **综合** | **26-39 分** | 🟠 偏弱~🔴 临界 |

### 校准要点
- 综合分应 <45（🟠 或 🔴）
- 微观分应 <20（几乎所有微观指标为负）
- 如果总分 >50，检查是否忽略了机构减持或 Alpha 持续为负的信号

---

## 自我校准流程

每轮分析 3 只以上基金后，检查自己的评分分布：

| 检查项 | 期望结果 |
|--------|---------|
| 最高分与最低分之差 | >25 分（有区分度，不是全部中性） |
| 🟢 和 🔴 比例 | 在 3+ 只基金组合中通常不会全部同色 |
| 评分与操作结论一致性 | 无「评分 <30 建议加仓」或「评分 >75 建议止损」的明显矛盾 |
| 宏观因子跨基金一致性 | 同一时间分析的多只基金，宏观分不应相差 >5 分 |

如出现偏差，重新校准。
