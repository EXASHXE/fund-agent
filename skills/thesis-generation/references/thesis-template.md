# Thesis Draft Template

`thesis_generation` produces draft narrative artifacts only.

## Template

1. Objective
2. Evidence used
3. Core thesis
4. Supporting facts
5. Counterpoints
6. Key uncertainties
7. Evidence gaps
8. Conditions to revisit
9. Escalation note for `decision_support`

## Chinese Example

```text
核心观点：当前组合的主要矛盾是权益和区域暴露偏集中，而不是单只基金短期涨跌。支持证据来自组合权重、风险标记和净值回撤。反方观点是部分高波动基金可能仍有长期配置价值，因此不应仅因浮亏卖出。若用户要求正式操作，需要进入 decision_support。
```
